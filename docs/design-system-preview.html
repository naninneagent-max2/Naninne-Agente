§ 2.4 — Cores por projeto

Escrita = azul (leitura). Audiovisual = laranja (luz/lente). Mercado = verde (crescimento). Tech = cinza-azulado (código). Escolhidas com luminância similar (55–65% L*) para nenhuma cor "gritar" mais que as outras.

⚠️ Regra de uso: cores 500 são para backgrounds, ícones grandes (≥24px) e gráficos. Para texto, sempre use a variante 700 — algumas 500 não passam WCAG AA em texto normal (3.58:1, 2.83:1).

§ 7 Animação

200ms é o ritmo padrão. Entradas ease-out, saídas ease-in, transições ease-in-out. Respeita prefers-reduced-motion.

Passe o mouse sobre os blocos para ver a animação:
